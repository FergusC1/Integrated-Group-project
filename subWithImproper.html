How to Subtract with Improper Fractions?

Subtracting with improper fractions is very similiar to adding with improper fractions. You follow the same steps except you have to subtract the numerators instead od add.

First we need to find the LCM for 7 and 3. This is covered in year 5. Click link to learn how to find the LCM and rationalise fractions.

The LCM for 7 and 3 is 21 so we need to make it so that both fractions denominator is 21.

Now we have the rationalised the fractions we can subtract the numerators. 27-24 is 13.
